# Decisiones de Producto

Un juego de cartas digital para practicar decisiones de UX y producto
a través de consecuencias acumuladas en métricas reales.

## Qué es
- Juego experimental en HTML / CSS / JS vanilla.
- Inspirado en juegos de mesa y cartas físicas.
- Cada decisión impacta conversión, soporte, satisfacción,
  salud del negocio y deuda UX.

## Estado
Prototipo jugable / en iteración.

## Por qué existe
Explorar cómo enseñar criterio de producto y UX
sin teoría pesada ni frameworks.
